# Derby-Memory-Game
## ゲーム開始方法
### IOS
1. iphoneでExpo Goをダウンロードする
2. ターミナルで*npx expo start*を実行
<br><img src="images/vscode-screen-shot.png" width="400px"><br><br>
3. QRを読み取り
4. iphoneで起動開始
### WEB
1. ターミナルで*npx run web*を実行
<br><img src="images/vscode-screen-shot.png" width="400px"><br><br>
2. QRを読み取り
3. WEBで起動開始
